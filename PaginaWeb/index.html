01 / Proyecto

¿Qué estamos
construyendo?

El anteproyecto consiste en crear un mini-datacenter de aula donde los estudiantes de SMR ponen en práctica conocimientos reales de infraestructura IT sobre hardware físico.

Diseñamos la red, montamos los servidores, configuramos los servicios y aseguramos el sistema. De la teoría al rack, paso a paso.

Redes Locales S.O. en Red Seguridad Informática Servicios de Red Aplicaciones Web Montaje y Mant.
◈

Diseño de red

Topología, direccionamiento IP, subredes y dispositivos.

◈

Montaje físico

Instalación de servidores, switches, routers y cableado.

◈

Configuración de S.O.

Windows Server y Linux según los servicios requeridos.

◈

Servicios de red

DNS, DHCP, FTP, correo y gestión de usuarios.

◈

Aplicaciones web

CMS/LMS integrado con correo y transferencia de archivos.

◈

Seguridad y backup

Cortafuegos, control de acceso y copias de respaldo.

02 / Servicios

Stack de
infraestructura

⬡

DNS / DHCP

Resolución de nombres y asignación dinámica de IPs en la red local.

⬡

FTP Server

Transferencia de archivos segura entre clientes y servidores del datacenter.

⬡

Mail Server

Servicio de correo electrónico interno integrado con el resto de la infraestructura.

⬡

NAS Storage

Servidor de almacenamiento en red centralizado para todos los equipos del aula.

⬡

CMS / LMS

Plataforma web colaborativa de aprendizaje desplegada sobre la infraestructura propia.

⬡

Firewall & Backup

Políticas de acceso, cortafuegos y copias de seguridad automatizadas.
